The top 5 fastest operating systems to restore old computers!

So everyone has that ancient computer that takes 5 hours to start. But you can make it Usain Bolt (or faster than it already is) by just installing a lightweight Linux operating system!
Today I will be making a list of the top 5 fastest operating systems.

5. Ubuntu

Ubuntu is the balance between performance, looks, and intuition. It is perfect but isn't as lightweight as other operating systems on this list. For a similar operating system, that is much faster but compromises looks, use:

4. Lubuntu

If that wasn't a smooth transition then i don't know what is.

Lubuntu is an operating system based off of Ubuntu. Its fast and that's about it.

3. Damn Small Linux

Damn small linux (DSL For short) is one of the smallest (ITS IN THE NAME) linux installations. It can even fit on 50 MB! It completly runs in ram so you can install it on a thumb drive and while working, you dont need to worry about your USB getting disconnected*. (* = programs may hang if usb disconnected, OS will work fine).

2. Puppy linux

Puppy linux is designed to be extremely lightweight. It can fit on 100 MB. Other than that and the looks, its almost the same as DSL.

And finally, 1. Linux lite

You might not use DSL or Puppy Linux for a very long, but you might use Linux Lite for a long time. It makes switching from Windows to Linux easy, is intuitive and looks and feels good.
