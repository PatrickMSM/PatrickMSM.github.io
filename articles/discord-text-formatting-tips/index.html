Do you want to know how to format your discord messages and make you seem like a pro? Find out now!

1. Bold text

To make bold text, insert two asterisks (∗∗) before the text you want to make bold and place two after them.


2. Italic text

To make some text italic, type an underline (_) before the text you chose and another one after the text. Or, type a single asterisk (∗) before and after the text you picked.


3. Underline text

To underline a piece of text, put two underlines (__) before and after the section of text.


4. Code

You can make code blocks by placing three backticks/backprimes (‵‵‵) before and after the text. You can also select a programming language for syntax highlighting! Just type a language name (ex. css, js, etc.) after the first three backticks and insert a newline after the language name!
